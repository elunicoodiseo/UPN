Módulo 2: Tokenización Visible

Ahora el modelo divide el texto en partes pequeñas.

La inteligencia artificial apoya la educación
¿Qué es un token? El modelo no procesa las frases igual que una persona. Primero las divide en unidades manejables llamadas tokens. Toca una ficha para ver más.

Para el LLM, las palabras se convierten en representaciones numéricas (vectores). Con esas representaciones, el modelo detecta relaciones, contexto y patrones.

Módulo 3: Predicción de la Siguiente Palabra

El modelo no busca una respuesta exacta; calcula qué palabra podría continuar.

La IA en educación puede ayudar a…

El LLM calcula qué palabra podría continuar según el contexto. No elige una palabra porque tenga conciencia o intención. La elige porque, estadísticamente, parece adecuada.

personalizar — 45%
automatizar — 25%
evaluar — 18%
orientar — 8%
caminar — 4%

Módulo 4: Transformer y Atención

El modelo no lee las palabras como una lista simple. Observa relaciones entre palabras para construir sentido.

¿Qué es la atención? Los modelos modernos usan arquitectura Transformer. La atención permite que el modelo identifique qué partes del texto son más importantes para interpretar el contexto.

  • Palabras más relacionadas → líneas más brillantes
  • Palabras menos relacionadas → líneas más tenues

El buscador recupera información existente. El LLM genera texto nuevo. Por eso, usar un LLM como si fuera una fuente puede llevar a errores.

Módulo 6: Alucinaciones

La respuesta puede sonar convincente, pero igual debe verificarse.

Respuesta generada por IA

Según el Informe Global de IA Educativa 2025 de la Universidad Mundial de Tecnología, el 92% de colegios usa IA adaptativa.

Toca las partes resaltadas para identificar señales de alerta.

Versión responsable:

Para afirmar que la IA adaptativa se usa ampliamente en colegios, primero se debe verificar la existencia de la fuente, revisar estudios académicos reales y contrastar el dato con más de una fuente confiable.

¿Qué es una alucinación? Ocurre cuando el LLM genera información falsa o no comprobada, pero la presenta con seguridad y buena redacción. La calidad del lenguaje no garantiza la verdad del contenido.

5 ideas clave

1Un LLM genera texto probable, no verdades garantizadas.
2Trabaja con tokens, patrones y contexto.
3La arquitectura Transformer ayuda a relacionar palabras e ideas.
4Puede generar datos, citas o fuentes falsas (alucinaciones).
5Toda respuesta importante debe verificarse y triangularse.
